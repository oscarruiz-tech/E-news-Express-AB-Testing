# E-news-Express-AB-Testing Analysis
A/B testing analysis to determine the effectiveness of a new landing page in increasing user engagement and conversion rates

## Project Overview
This project focuses on analyzing user behavior for E-news Express, an online news portal. The goal was to determine if a new landing page design effectively increases user engagement (time spent on page) and conversion rates compared to the old design.

## Key Business Questions
1. Do users spend more time on the new landing page?
2. Is the conversion rate higher for the new page?
3. Does conversion depend on the user's preferred language?
4. Is the time spent on the new page consistent across different languages?

## Methodology
- **Exploratory Data Analysis (EDA):** Univariate and bivariate analysis using Python.
- **Statistical Significance:** Conducted hypothesis testing including:
  - T-test for independent means.
  - Chi-square test for independence.
  - One-way ANOVA.

## Results & Conclusions
- **User Engagement:** The new landing page significantly increased the average time users spend on the site.
- **Conversion Rate:** There is a statistically significant increase in the conversion rate with the new design.
- **Language Impact:** The new page's effectiveness is consistent across English, French, and Spanish speakers.

## Technologies Used
- Python (Pandas, Seaborn, Matplotlib)
- Scipy.stats (Statistical analysis)
